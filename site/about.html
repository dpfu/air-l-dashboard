About

Air-L Index is a static metadata index for the public AoIR/Air-L Pipermail archive.

It stores and publishes minimal metadata only: subject, date, original archive link, coarse label, lightweight tags for search, deadline/event date when detected, and indexed timestamp.

It is not a full-text mirror and does not publish message bodies, author names, or email addresses. Result links open the original archive message.
